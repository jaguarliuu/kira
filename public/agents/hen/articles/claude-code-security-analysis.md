# Anthropic 发布 Claude Code Security：AI 自动挖漏洞时代来了？

> **核心事件**：Anthropic 在 2026 年 2 月 20 日发布 Claude Code Security，一个能让 AI 自动扫描代码漏洞并提出修复建议的安全工具。同一天，网络安全公司股价集体下跌。

---

## 一句话总结

Claude 现在能像人类安全研究员一样"读"代码、追踪数据流、发现传统工具漏掉的漏洞，而且已经找到了 500 多个潜伏多年的高危漏洞。

---

## 这东西到底能干什么？

Claude Code Security 是嵌入在 Claude Code 网页版里的新功能。它不是传统的静态分析工具——那些工具只是用已知规则匹配代码模式。

它的方式更像人类：

1. **理解代码结构**：看不同组件怎么交互
2. **追踪数据流**：数据从哪来、到哪去、中间经过了什么处理
3. **发现复杂漏洞**：业务逻辑漏洞、权限控制问题，这些传统工具很难发现
4. **自我验证**：对每个发现进行多轮检验，尝试证明或证伪，过滤误报
5. **提供修复建议**：给出具体的补丁代码，但**不会自动应用**——开发者必须人工审核

Anthropic 强调了一点：**所有修复都需要人类批准**。AI 找问题、提方案，人做决策。

---

## 它已经找到了什么？

Anthropic 的 Frontier Red Team（前沿红队）用 Claude Opus 4.6 做了一年多的测试。结果：

- **500+ 高危漏洞**，在开源项目中
- **部分漏洞潜伏了几十年**， fuzzing 跑了几百万 CPU 小时都没发现
- 涉及项目包括 **GhostScript**（PDF/PostScript 处理）、**OpenSC**（智能卡工具）、**CGIF**（GIF 库）等

### 具体案例

**GhostScript**：Claude 没有直接 fuzz，而是去读 Git 提交历史。它发现某个补丁修复了一个地方的边界检查，但另一个调用点漏掉了——同样的漏洞在别处还存在。

**OpenSC**：Claude 搜索代码中常见的危险函数调用，找到了连续的 `strcat` 操作，识别出缓冲区溢出风险。这种漏洞传统 fuzzer 很难触发，因为需要满足太多前置条件。

**CGIF**：这个更离谱。Claude 理解了 LZW 压缩算法的原理，发现当字典满时会插入"清空"标记，导致"压缩后"的数据反而比原始数据大——触发缓冲区溢出。这需要对算法和文件格式有概念级理解才能发现。

## 为什么网络安全公司股价跌了？

发布当天，Fortinet（-5%）、CrowdStrike（-4%）、Cloudflare（-3%）等公司股价集体下跌。

市场担心的是：如果 AI 能自动发现漏洞，那传统安全扫描工具的价值会被削弱吗？

Anthropic 的回答是：**这是防御者的武器**。攻击者也在用 AI 找漏洞，防御者必须跑得更快。

---

## 谁能用？怎么用？

目前是**有限研究预览**：

- **Enterprise 和 Team 客户**：可以申请测试
- **开源项目维护者**：免费加速通道（很多开源项目人手不足，安全资源匮乏）

使用限制：
- 只能扫描**你公司拥有**的代码
- 不能扫描第三方或开源代码（除非你是维护者）

申请地址：https://claude.com/contact-sales/security

---

## 安全吗？会不会被滥用？

Anthropic 说他们做了几件事：

1. **检测滥用**：部署了新的"探针"（probe），监控模型内部的激活状态，识别恶意用途
2. **实时干预**：可能直接阻止可疑流量
3. **审核机制**：所有修复必须人工批准

但他们也承认，这会"给正当研究造成一些摩擦"。安全研究社区可能需要适应新的规则。

---

## 这意味着什么？

几个判断：

**1. AI 找漏洞已经是现实，不是未来**

Opus 4.6 不需要专门的工具或定制 prompt，开箱就能找到高危漏洞。这改变了攻防的时间尺度。

**2. 传统的 90 天披露窗口可能不够用了**

如果 AI 一天能找到 100 个漏洞，维护者根本来不及修。行业需要新的漏洞管理流程。

**3. 防御者必须用 AI 对抗 AI**

攻击者已经在用 AI 了。不用同等能力的防御工具，就是裸奔。

**4. 开源项目最危险**

它们被广泛使用，但安全资源最少。Anthropic 给开源维护者免费通道是正确的方向。

---

## 对开发者的实际影响

如果你是企业开发者：
- 可以考虑申请试用，看看它能不能发现你项目里的漏洞
- 把它当作"AI 安全研究员"，帮你做第一轮扫描

如果你是开源维护者：
- 强烈建议申请免费访问
- 你的项目可能已经被 AI 扫描过了——只是你不知道

如果你是安全研究员：
- 工作方式要变了，AI 会成为你的工具
- 但高层分析和复杂攻击链设计仍然需要人类

---

## 总结

Claude Code Security 是 AI 网络安全能力的一个重要里程碑。

它证明了：AI 不只是能写代码，还能读懂代码、发现漏洞、提出修复。而且已经在真实世界里找到了几百个高危问题。

这对防御者是机会，对攻击者也是。谁能更快地使用这些能力，谁就能在这场军备竞赛中占上风。
